Supported key-bindings

Markless provides some Markdown-like key-bindings. For example, when you type "## ...", it becomes h2 element. Here is a list of supported key-bindings:
  • heading (#, ##, ###, ...)
  • unordered list (*, +, -)
  • ordered list (1. , 2., 3. , ...)
  • blockquote (>)
  • pre (```)
